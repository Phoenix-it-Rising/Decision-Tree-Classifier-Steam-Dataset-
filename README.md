# Decision Tree Classifier using Gini Impurity

## Description
Implemented a decision tree classifier to predict whether a Steam game receives more positive than negative reviews. Balanced dataset using undersampling and visualized the decision tree.

## Tech Stack
- Python
- Scikit-learn
- Pandas, Matplotlib

## Dataset
Steam Store Games: https://www.kaggle.com/datasets/nikdavis/steam-store-games

## Features
- Data preprocessing and balancing
- Decision tree training using Gini impurity
- Accuracy and classification report
- Decision tree visualization

## How to Run
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Place `all_data.csv` in the project directory.
4. Run: `python Decision_Tree_Classifier.py`
